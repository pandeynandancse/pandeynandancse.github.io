Project : Coronavirus cases Update

Coronavirus Latest Notification (For India)

Project is developed in python using flask.
First it scrapes the ministry of home website to get real time data once application is opened.
After that say 'state name' say you want to know about uttar pradesh then speak 'Uttar Pradesh'.
It will tell you about different categories of cases of that state by speaking.
If you want to stop the project say 'stop'. Application will exit.
