Project : MNIST

Multiclass Classification

Two Datasets : train.csv , test.csv
Link : Link to Dataset      Link to Kaggle Notebook

Category : Classification
Steps :
a) Load dataset using Pandas ==>> pixels of each image is given row-wise & no. of rows tells no. of images.
b) Seperate label column from train dataset
c) Re-scale train and test image from [0...255] to [0..1] ==>> train/255 and test/255.
d) Resize train nd test image into three dimensions from one dimension ==>> (-1, 28,28,1).
e) Above step means each image's height and width is of now 28 and channels is 1.
f) Now convert training label into categorical values (one hot encoding) using tensorflow's to_categorical() method.
g) Training and validation split of features and label using sklearn.model_selection import train_test_split.
h) Create convolution nueral network and select proper optimizer with some learning rate and then compiler your model using categorical crossentropy.
i) Set learning rate annealer i.e. ReduceLRonPlateau so that if no improvement is seen in monitorng metrics LR can be decreased by some factor.
j) For image data augmentation of training data use ImageDataGenerator (Note: Augmentation is not done on validation or test dataset).
k) Now fit model using fit_generator and take data using flow() or flowfromdataframe() or flowfromdirectory() according to location of dataset.
l) Lastly predict on test set.
