Project : Auto Mail Sender

Send mail to multiple people via python

A web application that allows users to send mail to multiple people at same time.
Data is stored locally. So need to worry about privacy.
Email name and password of sender is stored in 'users.xlsx' and the mail id of receivers is stored in 'mails.xlsx' file.
Whenever you want , you can delete that file/s.
