Project : Next Word Prediction

Predict Next Word Using BERT

Developed using Streamlit and Pytorch


It uses the state of Art NLP Model BERT to predict next word. WebPage has been developed using Streamlit. In this webapp extra feature is that XLNET model is also given so that one can see difference between both model's prediction. One can add more STATE of ART NLP models also.
